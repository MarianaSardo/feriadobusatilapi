Documentación de la API

Información completa sobre los endpoints y uso de la API

📋 Información General

URL Base: http://localhost:8000
Documentación Swagger: Ver Swagger UI
Formato de Respuesta: JSON

🔍 Endpoints de Consulta (Sin Autenticación)

GET /

Información general de la API

GET /feriados/all

Todos los feriados de todos los años

GET /feriados/{anio}

Feriados de un año específico

GET /feriados/consultar/{fecha}

Verificar si una fecha es feriado (YYYY-MM-DD)

GET /feriados/proximos?cantidad=5

Próximos feriados (cantidad opcional: 1-20)

🔧 Endpoints de Administración (Requiere API Key)

POST /feriados/agregar/

Agregar un nuevo feriado

Headers: X-API-Key: tu_api_key
Parámetros: anio, fecha, nombre
DELETE /feriados/eliminar/

Eliminar un feriado existente

Headers: X-API-Key: tu_api_key
Parámetros: anio, fecha

💡 Ejemplos de Uso

Obtener feriados de 2024

GET /feriados/2024

Consultar fecha específica

GET /feriados/consultar/2024-12-25

Próximos 10 feriados

GET /feriados/proximos?cantidad=10

Agregar feriado (con API Key)

POST /feriados/agregar/
Headers: X-API-Key: tu_api_key
Body: anio=2024&fecha=2024-12-25&nombre=Navidad

📄 Ejemplos de Respuesta

Consulta de fecha

{
  "fecha": "2024-12-25",
  "es_feriado": true,
  "nombre": "Navidad"
}

Próximos feriados

[
  {
    "fecha": "2024-12-25",
    "nombre": "Navidad",
    "dias_restantes": 15
  }
]
